## Golden Paths (Customer Journeys)

These are the primary UX flows on this site:

| # | Path | Entry Point | End State |
|---|------|-------------|-----------|
| 1 | Storage Browsing | Homepage → Storage page → Filter → Select | Redirect to bridge-ai for rental |
| 2 | Space Booking | Homepage → Spaces → Calendar → Book | Application submitted via API |
| 3 | Event RSVP | Homepage → Events → Browse → RSVP | Signup confirmed via API |
| 4 | Discovery | Homepage → Identity cards → Section pages | Visitor engaged |
| 5 | Portal Access | Nav "Sign In" | Redirect to bridge-ai portal |

Note: Payment, onboarding, and portal flows happen on bridge-ai, not this site.

## DO NOT

- Use inline `style=""` attributes
- Use Bootstrap or other framework classes
- Create custom CSS when Tailwind utilities suffice
- Use non-brand colors for primary UI elements
- Use Gotham font (licensing — use Inter)
- Hardcode API URLs (use js/config.js)
- Build payment forms (Stripe stays in bridge-ai)
